# Critical Analysis: Is $(e^{i\pi})^{2^n}=1$ the Bott Periodicity Ladder?

Companion note to `DepthGradedClosure.lean`.
Pinnacle Quantum Group — April 2026.

## 1. Claim under review

The depth-graded extension posits an infinite tower of rotational closures
indexed by $n\in\mathbb{N}$:

$$
(e^{i\pi})^{2^n} \;=\; \hat 1_n
$$

with the readings

| rung $n$ | value              | claimed interpretation                          |
|:--------:|--------------------|-------------------------------------------------|
| $0$      | $(e^{i\pi})^1=-1$  | Euler half-closure on $U(1)$                    |
| $1$      | $(e^{i\pi})^2=1$   | full-rotation closure on $U(1)$                 |
| $2$      | $(e^{i\pi})^4=1$   | spinor closure on $SU(2)$ — Dirac's $4\pi$      |
| $3$      | $(e^{i\pi})^8=1$   | "octonion-scale closure"                        |
| $n$      | $(e^{i\pi})^{2^n}$ | "Bott periodicity ladder"                       |

## 2. What is literally true

The formula is a theorem — an elementary one. In $\mathbb{C}^\times$,
$e^{i\pi}=-1$ is of order two, so $(-1)^{2^n}=1$ for every $n\ge 1$ and
$(-1)^{2^0}=-1$. This is exactly the content of `DepthGradedClosure.lean`:

- `depth_zero            : depthClosure 0 = -1`
- `depth_closure_pos     : 1 ≤ n → depthClosure n = 1`
- `depth_closure_stable  : ∀ n m ≥ 1, depthClosure n = depthClosure m`

All three are closed with no `sorry`.

## 3. Where the analogy bends

**Rung 0 and rung 1** are Euler's identity and its square. Nothing to dispute.

**Rung 2** is genuine — but subtler than the formula suggests.
The double cover $\mathrm{Spin}(3)=SU(2)\twoheadrightarrow SO(3)$ is the
*first and only* double cover of a rotation group by a connected Lie group.
In $SU(2)$ the element $\exp(i\pi\,\hat n\!\cdot\!\vec\sigma)=-I_2$ has order
two, so two half-rotations (total $4\pi$) return the identity. The formula
$(e^{i\pi})^4=1$ captures this *only through the abelian phase*: it is the
computation inside the $U(1)$ subgroup generated by $-I_2$. The genuine
spinor content — that a $2\pi$ rotation of a fermion is $-1$ — lives in the
non-trivial $\pi_1(SO(3))=\mathbb{Z}/2$, which the scalar identity does
not see. In the Lean file we record the abelian witness as
`spin_two_return : ((-1 : \mathrm{Matrix}\,(\mathrm{Fin}\,2)\,(\mathrm{Fin}\,2)\,\mathbb{C}))^2 = 1`.

**Rung $n\ge 3$ is where the ladder breaks.** Three separate phenomena get
conflated:

1. **The exponent $2^n$.** The formula $(-1)^{2^n}=1$ is true for every
   $n\ge 1$, but stabilises immediately at rung 1. Rungs 2,3,4,… carry the
   same value and no new $U(1)$ information. `depth_closure_stable` makes
   this collapse explicit.

2. **Further "double covers."** Beyond $SU(2)=\mathrm{Spin}(3)$, there is no
   canonical tower of connected-Lie-group double covers to climb. For all
   $m\ge 3$ the group $\mathrm{Spin}(m)$ is already simply connected
   ($\pi_1=0$), so it has no non-trivial connected covering group. The
   phrase "each depth level adds a double cover" is therefore not backed by
   a Lie-group construction at rung 3 or higher.

3. **Octonions at rung 3.** The 7-sphere $S^7$ of unit octonions is not a
   Lie group: octonionic multiplication is non-associative. There is no
   "$8\pi$ octonionic rotation closure" in the sense that $4\pi$ is a
   closure in $SU(2)$. One can speak of the Hopf fibration
   $S^3\hookrightarrow S^7\twoheadrightarrow S^4$, but this is a fibre
   bundle, not a cover, and it is not an iteration of the double cover
   $SU(2)\twoheadrightarrow SO(3)$.

4. **Bott periodicity.** The actual Bott theorem gives the homotopy groups
   of the stable unitary/orthogonal groups with *period 2* (complex) or
   *period 8* (real). The period is a single fixed integer, not a function
   of depth $n$, and the homotopy groups take specific values $(\mathbb Z,
   \mathbb Z/2, 0, \ldots)$. The exponent $2^n$ in the RLA ladder does not
   recover these periodic homotopy groups: matching the real period 8 at
   rung 3 is numerical coincidence, not derivation.

The dimension sequence $1,2,4,8$ of the normed division algebras
$\mathbb R,\mathbb C,\mathbb H,\mathbb O$ does follow the
Cayley–Dickson doubling pattern (Hurwitz's theorem), and *that* pattern
halts at $\mathbb O$: the sedenions at dimension $16$ have zero divisors
and are not a division algebra. A genuine rung 4 does not exist in this
sequence either.

## 4. Summary verdict

- **Rung 0:** Euler's identity — correct, trivially formalised.
- **Rung 1:** $(-1)^2=1$ — correct, trivially formalised.
- **Rung 2:** Formally captures the $4\pi$ return inside the $U(1)$ phase
  subgroup of $SU(2)$. The fuller spinor statement requires
  $\pi_1(SO(3))=\mathbb Z/2$, which the formula does not supply.
- **Rung $n\ge 3$:** $(e^{i\pi})^{2^n}=1$ is true but vacuous (identical to
  rung 1). The interpretation as "higher double covers,"
  "octonion closure," or "Bott periodicity" is an *analogy*, not a
  theorem. Each invocation refers to a genuine but *different*
  mathematical structure (Hopf fibrations, Cayley–Dickson doubling,
  stable homotopy periodicity); none of them is the identity
  $(e^{i\pi})^{2^n}=1$.

## 5. Honest formal deliverable

`DepthGradedClosure.lean` proves exactly what the underlying identity
yields, and no more. It contains:

- the definition `depthClosure n := (e^{iπ})^{2^n}`,
- `depth_zero`, `depth_closure_pos`, `depth_closure_dichotomy`,
  `depth_closure_norm`, `depth_closure_stable`,
- the $2\times 2$ witness `spin_two_return`.

All six theorems are closed without `sorry`. The docstring flags which
parts of the intended narrative are *not* captured, so that downstream
readers — in particular the physics audience that takes the ladder at
face value — are not misled.

## 6. What a rigorous ladder would need

If one wanted to turn the analogy into mathematics, the minimal
commitments are:

1. A formal definition of the depth-$n$ "rotation group" $G_n$ whose
   generator is supposed to satisfy $g^{2^n}=e$.
2. A construction $G_n\twoheadrightarrow G_{n-1}$ that is a group
   homomorphism with kernel $\mathbb{Z}/2$ (a genuine double cover).
3. A proof that the given closure order $2^n$ is the *smallest* integer
   $k$ with $g^k=e$ — otherwise the ladder collapses as in
   `depth_closure_stable`.

For $n=0,1,2$ (classical $\mathbb Z/2$, $U(1)$, $SU(2)\twoheadrightarrow
SO(3)$) all three hold. For $n\ge 3$, (2) fails: there is no connected
Lie group that double-covers $\mathrm{Spin}(m)$ for $m\ge 3$, hence no
object to put at rung 3 without changing the rules (e.g. passing to
groupoids, 2-groups, or higher gerbes — each of which is a legitimate
direction but a different theorem).

Formalising the non-existence result requires
$\pi_1(\mathrm{Spin}(m))=0$ for $m\ge 3$, which is well outside the
current Mathlib coverage of Lie-group topology. We therefore do not
attempt it here, and we do not claim it in `DepthGradedClosure.lean`.
